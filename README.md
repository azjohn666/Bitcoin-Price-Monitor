# Bitcoin-Price-Monitor
This is a Bitcoin Price Monitor written in Python. This is still a working project.


# Dependencies

- numpy
- matplotlib
- requests


# Installation

```
$ cd
$ git clone https://github.com/azjohn666/Bitcoin-Price-Monitor.git
$ cd <MAIN_DIRECTORY>
```



# Usage

```
$ cd <MAIN_DIRECTORY>
$ python3 example/run_Monitor.py
```
